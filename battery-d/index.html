#
auto getBatteryList();
Gets list of all aviable batteries in system
Examples: 
import std.array;
import battery.d;
getBatteryList().array; // ["BAT0"], laptops in general have only 1 battery
getBatteryList().front; // "BAT1"
Returns:
range of battery names
