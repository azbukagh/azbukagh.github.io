Function SemVer.nextMajor

Next Major/Minor/Patch version Increments version in semver way

Prototype

SemVer nextMajor() pure nothrow @nogc @safe;

Example

1.2.3 -> nextMajor -> 2.0.0 1.2.3 -> nextMinor -> 1.3.0 1.2.3 -> nextPatch -> 1.2.4 1.2.3-rc.1+build.5 -> nextPatch -> 1.2.4
